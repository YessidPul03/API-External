Editar Petición (Posts)

Está petición se utiliza para actualizar / editar por completo un registro existente, es decir, es parecido a realizar un UPDATE a la base de datos.

Editar

Eliminar Petición (Post)

Esta petición se utiliza para eliminar un registro existente, es similar a un DELETE a la base de datos.

Eliminar

Crear Petición (Post)

Está petición es utilizada para solicitar la creación de un nuevo registro, es decir, algo que no existía previamente. Esto es equivalente a realizar un INSERT en la base de datos.

Crear
